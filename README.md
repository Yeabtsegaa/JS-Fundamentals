# 📘 JavaScript Fundamentals

Welcome to my JavaScript Fundamentals repository! This project contains well-organized examples and mini-projects that cover the core concepts of JavaScript. It's perfect for beginners looking to build a solid foundation in JS.

---


---

## 📚 Topics Covered

### ✅ Basics
- Variables (`let`, `const`, `var`)
- Data Types
- Operators

### ✅ Control Structures
- If/else statements
- Loops (`for`, `while`, `do...while`)
- Switch statements

### ✅ Functions
- Function declarations & expressions
- Arrow functions
- Callback functions

### ✅ Arrays & Objects
- Array methods (`map`, `filter`, `reduce`)
- Object manipulation

### ✅ DOM & Events
- Selecting and modifying DOM elements
- Event listeners and event handling

### ✅ ES6+ Features
- Destructuring
- Spread and Rest operators
- Template literals
- Modules (import/export)

### ✅ Asynchronous JS
- setTimeout & setInterval
- Promises
- Async/Await
- Fetch API (simple API examples)
